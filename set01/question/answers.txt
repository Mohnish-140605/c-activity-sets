Questions related to set01
What is main function used for?
It is used to call all the other functions in the program.

Where is the calling function and called function placed in the program?
The called function is placed inside the main function and the calling function is placed below or above the main function in the program.

What is call by reference?
It refers as we change the value of called function in formal arguments, the value of calling function in actual arguments changes.

What is call by value?
It refers as we change the value of called function in formal arguments, the value of calling function in actual arguments does not change.

Why are strings & arrays passed by CBV?
Because they store the value by themselves.

Can CBV and CBR be used in same program?
Yes, it can be used.

How is CBR executed?
They are executed by pointers and pointer variable.
